ACTO 2 · EL ANÁLISIS TÉCNICO

¿Qué es la Ingeniería Social?

Es el arte de manipular a las personas para que revelen información confidencial o realicen acciones que comprometen su seguridad.

En lugar de atacar máquinas, el delincuente ataca emociones y comportamientos humanos:

  • 🎭 Se hace pasar por alguien de confianza (tu banco, un amigo, soporte técnico).
  • 🧲 Usa nuestras emociones como anzuelo.
  • 🔓 Te convierte a TI en la "llave" que abre la puerta.
ACTO 2 · EL ALGORITMO

Un ataque no es "magia": es un algoritmo 💻

Todo ataque sigue los mismos pasos lógicos. Si conoces el algoritmo, puedes detenerlo.

// Algoritmo de un ataque de ingeniería social
function hackearHumano(victima) {
  1. setup   = crearDisfrazDeConfianza("tu banco");   // suplantar
  2. gatillo = activarEmocion("miedo", "urgencia"); // presionar
  3. accion  = pedir("clic / contraseña / dinero");    // engañar
  4. return ejecutar(accion);   // la víctima obedece
}

💡 Tu punto de defensa: si interrumpes cualquier paso (dudas del disfraz, ignoras el gatillo o no realizas la acción), el algoritmo falla.

ACTO 2 · LOS GATILLOS (INPUTS)

Paso 2 del algoritmo: los "gatillos" 🪤

El gatillo es el input emocional que dispara la acción. Estos son los más usados:

⏳

Urgencia

"¡Actúa AHORA o perderás tu cuenta!" — no te deja pensar.

👑

Autoridad

"Soy del banco / tu director / la policía." — obedecemos sin dudar.

😨

Miedo

"Detectamos un acceso sospechoso a tu cuenta." — el pánico nubla.

🎁

Codicia / Curiosidad

"Ganaste un premio" / "Mira esta foto tuya." — caemos por tentación.

ACTO 2 · LA IMPLEMENTACIÓN

🎣 Phishing: el algoritmo más usado

El phishing es la "implementación" más común de ese algoritmo. Viene de "fishing" (pescar): se lanza un anzuelo (mensaje falso) esperando que alguien "muerda".

El objetivo: que entregues contraseñas, datos bancarios o dinero, creyendo que el mensaje es real.

📧

Correo falso

El más común.

🔗

Enlace trampa

Lleva a una web clonada.

📎

Archivo infectado

Instala malware al abrirlo.

VARIANTES

El phishing tiene "primos"

📱

Smishing

Phishing por SMS o mensaje de texto. "Tu paquete está retenido, paga aquí."

📞

Vishing

Por llamada de voz. "Hablamos del banco, confirme su NIP."

🎯

Spear Phishing

Ataque personalizado a una persona concreta, usando sus datos reales.

🐋

Whaling

Va por "peces gordos": directores y jefes con acceso a mucho dinero.

MÁS TRUCOS

Otras técnicas de ingeniería social

🎭

Pretexting

Inventan una historia creíble ("soy de soporte TI") para ganar tu confianza.

🪤

Baiting (cebo)

Dejan un USB "perdido" o una descarga gratis con malware dentro.

🚪

Tailgating

Te siguen para colarse a un lugar restringido: "¿me detienes la puerta?"

🤝

Quid pro quo

"Te doy algo a cambio de algo": ayuda falsa a cambio de tus datos.
